# Network File Transfer System

This project demonstrates a basic Java client-server file transfer system using TCP and UDP sockets.

The TCP version uses Java sockets for reliable file transfer. The UDP version implements a basic Stop-and-Wait style approach using acknowledgments and timeouts to explore how reliability can be added on top of UDP.

## Technologies
- Java
- TCP/IP
- UDP
- Sockets
- File I/O

## What I Learned
- How client-server communication works
- How files can be sent over sockets
- The difference between TCP and UDP
- How acknowledgments and timeouts can improve reliability
- Basic network troubleshooting concepts
